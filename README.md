# bookbot

## Description
Parses a file and extracts the word count and frequency of each letter. Then prints to the console the letter and frequncy in decending order.

## "Why?"
Decryting a simple substitution cypher can be done using frequency analysis (our first known record is from Al-Kindi ~9th century). By counting the frequency of each letter and substituting with the matching distribution. This tool automates retreiving the frequency of letters in the text.

## Quick Start (how)
- make sure to have python 3 or later.
- download the main.py source file.

## Usage
Open a terminal to the directory containing the main.py file.

```
python main.py
```
In some cases you might need to put in.

```
python3 main.py
```
then follow the prompt.
